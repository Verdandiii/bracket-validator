# Система проверки скобочных последовательностей

Валидатор синтаксической корректности вложенных конструкций с поддержкой многосимвольных скобок.

## Возможности
- Поддержка коротких скобок: `(`, `)`, `{`, `}`, `[`, `]`, `<`, `>`
- Поддержка многосимвольных скобок: `begin...end`, `/*...*/`
- Валидация корректности вложенности и парности скобок
- Обработка краевых случаев: пустая строка, непарные скобки

## Стек технологий
- **Язык:** Java 25
- **Структуры данных:** Собственная реализация стека (обобщённый класс `Stack<T>` на основе массива)
- **Тестирование:** JUnit 5
- **Коллекции:** HashMap, List

## Архитектура решения
Алгоритм посимвольно обходит строку и проверяет совпадение с префиксами открывающих и закрывающих скобок. При встрече открывающей скобки она помещается в стек, при встрече закрывающей — проверяется соответствие верхнему элементу стека.

## Запуск и тестирование

### Требования
- JDK 17 или выше
- JUnit 5 (для тестов)
